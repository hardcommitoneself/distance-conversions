## Installation

You can install the package via composer:

```bash
composer require jeffwhansen/distance-conversions
```

## Usage

```php
$skeleton = new Jeffwhansen\DistanceConversions();
echo $skeleton->echoPhrase('Hello, Jeffwhansen!');
```
